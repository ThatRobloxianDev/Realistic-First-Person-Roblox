# Realistic First-Person Roblox

A lightweight and simple system that improves the **first-person experience in Roblox** by making camera movement and character behavior feel more realistic and immersive.

---

## Features
- Smooth and realistic first-person camera  
- Better character alignment and movement feel  
- Lightweight and performance-friendly  
- Easy plug-and-play setup (no extra dependencies)  
- Works with default Roblox character system
